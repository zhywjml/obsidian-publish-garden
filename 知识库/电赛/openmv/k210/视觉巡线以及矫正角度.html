      • 一开始思路错了，好久没看，忘了，认为要等分六个区域
        • 一开始使用的是img.crop()将其分割为等大的六个小区域
        • 然而调用第二个就报错了，具体报错信息说的是边界超出范围，事实测试并没有
        • 后来用Range划定块状区域，等分成了六个区域
        • 结果发现判断出问题了，六个等分区域怎么可能区分出T形或者十字形
      • 回去看了一遍，按照图片思路参考操作一下划分区域
        • 区域具体划分见：区域划分技巧
        • 设置标志位，来通过与或判断。
          • 例如中间三个区域判断出来了，通过与运算，就可以判断是一条直线。
          • 其它也就同理了。

import sensor #摄像头对象，通过import直接调用

import image #

import lcd

import time

  

  

#初始化镜头

sensor.reset() #摄像头初始化

lcd.init(freq=15000000)

sensor.set_hmirror(1) #设置水平镜像 1-开启

sensor.set_vflip(1) #设置垂直翻转 1-开启

sensor.set_pixformat(sensor.RGB565) #pixformat 像素格式设置 sensor.GRAYSCAL-灰度图像 sensor.RGB565-彩色图像

sensor.set_framesize(sensor.QVGA) #设置每帧大小 sensor.QVGA-320*640

sensor.skip_frames(time = 2000)     # 等待设置生效.

clock = time.clock()                # 创建一个时钟来追踪 FPS（每秒拍摄帧数）

sensor.set_auto_whitebal(False)

  

Range0 = (110,0,100,30)

Range1 = (110,100,100,20)

Range2 = (110,210,100,30)

Range3 = (40,60,40,100)

Range4 = (240,60,40,100)

  

height = 240

Range_1_Part = (0,0,320,height//6)

Range_2_Part = (0,height//6,320,height//6)

Range_3_Part = (0,height*2//6,320,height//6)

Range_4_Part = (0,height*3//6,320,height//6)

Range_5_Part = (0,height*4//6,320,height//6)

Range_6_Part = (0,height*5//6,320,height//6)

  

font_scale = 2  # 字体大小

text_color = (255, 255, 255)  # 白色文本

bg_color = (0, 0, 0)  # 黑色背景

text01 = "xx"

  

while(True):

    clock.tick()                    # 更新 FPS 时钟.

    img = sensor.snapshot()         # 拍摄一个图片并保存.

    #lcd.clear()  # 清除屏幕

    #初始化

    # 初始化检测标志

    blob_in_range0 = blob_in_range1 = blob_in_range2 = False

    blob_in_range3 = blob_in_range4 = blob_in_range5 = False

    for blob in img.find_blobs([(0, 45, -46, 20, -128, 127)],roi=Range2,pixels_threshold=100,merge=True,margin = 10):

        img.draw_rectangle(blob.rect())

        img.draw_cross(blob.cx(),blob.cy())

    for blob in img.find_blobs([(0, 45, -46, 20, -128, 127)],roi=Range_2_Part,pixels_threshold=100,merge=True,margin = 10):

        #img.draw_rectangle(blob.rect())

    for blob in img.find_blobs([(0, 45, -46, 20, -128, 127)],roi=Range_3_Part,pixels_threshold=100,merge=True,margin = 10):

        #img.draw_rectangle(blob.rect())

    for blob in img.find_blobs([(0, 45, -46, 20, -128, 127)],roi=Range_4_Part,pixels_threshold=100,merge=True,margin = 10):

        #img.draw_rectangle(blob.rect())

区域该怎么找

这里，采用简单的色彩阈值法，毕竟读取到的图像只有白色黑色两种颜色，通过色彩阈值是适合的。
软件中也提供了方便的工具：阈值编辑器


打开后选择数据缓冲区域读取数据，就是摄像头拍摄到的图片会作为参考进行处理。
所以，先通过代码初始化将图像传入电脑，下面只是演示。

  • 参数怎么调整呢？
    • 好问题，重置滑块后二进制图像会变为纯白。
    • 首先拖动一个滑块（注意，只是拖动一个，类似于控制变量法）。
    • 先把想要保留的部分，变成白色。
    • 在拖动其他的，使白色与你想保留的部分，尽可能多的重叠。
    • 然后就可以复制LAB阈值了

如果调整后不太合理，想要重新开始，就点击：重置滑块。即可
