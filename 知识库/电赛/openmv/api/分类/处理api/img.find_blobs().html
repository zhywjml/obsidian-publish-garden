img.find_blobs()

在OpenMV中，img.find_blobs() 函数用于在图像中查找“blobs”，即二值图像中的连通区域。这些区域通常表示为物体或特征，其颜色与周围环境形成对比。这个函数对于物体检测、颜色识别和图像分析非常有用。

以下是一些关于 img.find_blobs() 函数的基本用法：

  1. 函数定义:

    blobs = img.find_blobs(thresholds, roi=(0, 0, img.width(), img.height()), x_stride=2, y_stride=2, invert=False, area_threshold=10, pixels_threshold=10, merge=True)
    • thresholds: 颜色阈值，用于定义哪些像素属于blob。可以是单个的(min, max)元组，也可以是多个元组的列表。
    • roi: 感兴趣区域（Region of Interest），定义了搜索blobs的图像区域。
    • x_stride: 搜索时在x方向上的步长。
    • y_stride: 搜索时在y方向上的步长。
    • invert: 是否反转阈值。
    • area_threshold: 过滤掉面积小于这个值的blobs。
    • pixels_threshold: 过滤掉像素数量小于这个值的blobs。
    • merge: 是否合并相邻的blobs。

  2. 返回值:

    • 函数返回一个blobs列表，每个blob都是一个包含多种属性的字典，例如：
      • x, y: blob的中心坐标。
      • w, h: blob的宽度和高度。
      • angle: blob的主要方向。
      • kind: blob的类型（圆形、矩形等）。
      • pixels: blob的像素列表。

  3. 示例代码:

    import sensor

sensor.reset()  # 重置摄像头
sensor.set_pixformat(sensor.RGB565)  # 设置像素格式
sensor.set_framesize(sensor.QVGA)  # 设置帧大小

while True:
    img = sensor.snapshot()  # 捕获图像
    img.binary([150, 150, 150], [200, 200, 200])  # 将图像二值化
    blobs = img.find_blobs([200])  # 寻找blobs

    for blob in blobs:
        img.draw_rectangle(blob['x'], blob['y'], blob['w'], blob['h'])  # 绘制矩形框

  4. 其他信息:

    • img.find_blobs() 可以用于多种场景，包括颜色识别、物体计数、形状分析等。
    • 可以通过调整参数来优化blobs的检测，例如通过改变阈值、步长、面积和像素阈值。

  5. 注意:

    • 在使用 img.find_blobs() 之前，通常需要先将图像二值化，以便更容易地识别连通区域。
    • 根据应用场景的不同，可能需要调整参数以获得最佳结果。

通过使用 img.find_blobs() 函数，你可以在OpenMV项目中实现对图像中特定颜色或形状的物体进行检测和分析。

blobs是一个列表

find_blobs对象返回的是多个blob的列表。（注意区分blobs和blob，这只是一个名字，用来区分多个色块，和一个色块）。
列表类似与C语言的数组，一个blobs列表里包含很多blob对象，blobs对象就是色块，每个blobs对象包含一个色块的信息。

blobs = img.find_blobs([red])

blobs就是很多色块。

可以用for循环把所有的色块找一遍。

for blob in blobs:
    print(blob.cx())

对于for循环的使用，见python背景知识

blob色块对象

blob有多个方法：

  • blob.rect() 返回这个色块的外框——矩形元组(x, y, w, h)，可以直接在image.draw_rectangle中使用。
  • blob.x() 返回色块的外框的x坐标（int），也可以通过blob[0]来获取。
  • blob.y() 返回色块的外框的y坐标（int），也可以通过blob[1]来获取。
  • blob.w() 返回色块的外框的宽度w（int），也可以通过blob[2]来获取。
  • blob.h() 返回色块的外框的高度h（int），也可以通过blob[3]来获取。
  • blob.pixels() 返回色块的像素数量（int），也可以通过blob[4]来获取。
  • blob.cx() 返回色块的外框的中心x坐标（int），也可以通过blob[5]来获取。
  • blob.cy() 返回色块的外框的中心y坐标（int），也可以通过blob[6]来获取。
  • blob.rotation() 返回色块的旋转角度（单位为弧度）（float）。如果色块类似一个铅笔，那么这个值为0~180°。如果色块是一个圆，那么这个值是无用的。如果色块完全没有对称性，那么你会得到0~360°，也可以通过blob[7]来获取。
  • blob.code() 返回一个16bit数字，每一个bit会对应每一个阈值。举个例子： blobs = img.find_blobs([red, blue, yellow], merge=True)

如果这个色块是红色，那么它的code就是0001，如果是蓝色，那么它的code就是0010。注意：一个blob可能是合并的，如果是红色和蓝色的blob，那么这个blob就是0011。这个功能可以用于查找颜色代码。也可以通过blob[8]来获取。

  • blob.count() 如果merge=True，那么就会有多个blob被合并到一个blob，这个函数返回的就是这个的数量。如果merge=False，那么返回值总是1。也可以通过blob[9]来获取。
  • blob.area() 返回色块的外框的面积。应该等于(w * h)
  • blob.density() 返回色块的密度。这等于色块的像素数除以外框的区域。如果密度较低，那么说明目标锁定的不是很好。
    比如，识别一个红色的圆，返回的blob.pixels()是目标圆的像素点数，blob.area()是圆的外接正方形的面积。
