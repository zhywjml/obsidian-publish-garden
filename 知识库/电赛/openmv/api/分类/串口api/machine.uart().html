machine.UART()

machine.UART(uart,baudrate,bits,parity,stop,timeout, read_buf_len)

构造函数​

machine.UART(uart,baudrate,bits,parity,stop,timeout, read_buf_len)

创建UART对象。

  • uart ：串口编号，共3个可用：
    • UART.UART1: 串口1；
    • UART.UART2: 串口2；
    • UART.UART3: 串口3；
  • baudrate：波特率，常用115200、9600;
  • bits ：数据位，默认8;
  • parity ：奇偶校验，默认None;
    • UART.PARITY_EVEN: 偶校验；
    • UART.PARITY_ODD: 奇校验；
  • stop: 停止位，支持 1， 1.5, 2， 默认 1 。
  • timeout: 串口接收超时时间。
  • read_buf_len: 串口接收缓冲大小。

在OpenMV中，machine.UART 是一个类，用于初始化和控制UART（通用异步收发传输器）接口，实现串行通信。以下是 machine.UART 类的一些基本用法和参数说明：

  1. 参数说明:

    • uart: 指定UART端口，例如 machine.UART.UART0 或 machine.UART.UART1。
    • baudrate: 波特率，表示每秒传输的比特数。
    • bits: 数据位，通常是7或8。
    • parity: 校验位，可以是 None（无校验）、machine.UART.PARITY_EVEN（偶校验）、machine.UART.PARITY_ODD（奇校验）。
    • stop: 停止位，可以是1或2。
    • timeout: 超时时间，单位为秒。
    • read_buf_len: 接收缓冲区的长度。

  2. 函数定义:

    uart = machine.UART(uart, baudrate, bits, parity, stop, timeout, read_buf_len)

  3. 使用方式:

    • 调用 machine.UART 并传入相应的参数，以初始化UART。

  4. 示例代码:

    import machine

# 初始化UART
uart = machine.UART(
    uart=machine.UART.UART0,
    baudrate=9600,
    bits=8,
    parity=None,
    stop=1,
    timeout=1000,
    read_buf_len=64
)

# 发送数据
uart.write('Hello World\n')

# 接收数据
data = uart.read(10)  # 读取10个字节

  5. 其他信息:

    • UART是串行通信接口，常用于与外部设备（如传感器、其他微控制器等）进行通信。
    • 通过 write() 方法发送数据，通过 read() 方法接收数据。

  6. 注意:

    • 确保UART的配置参数（波特率、数据位、校验位、停止位）与通信的另一端设备匹配。
    • timeout 参数定义了在没有数据接收时的等待时间。
    • read_buf_len 定义了接收缓冲区的大小，根据需要接收的数据量进行调整。

使用 machine.UART 类，你可以在OpenMV项目中实现与其他设备的串行通信，进行数据的发送和接收。
